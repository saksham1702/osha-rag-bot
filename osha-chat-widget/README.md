# OSHA Safety Chat Widget - WordPress Plugin

AI-powered floating chat widget that answers OSHA workplace safety questions using RAG technology with 500+ regulation pages.

## Installation

1. Download `osha-chat-widget.zip` from `/Users/pookie/Downloads/Knowella_osha/`
2. In WordPress admin: **Plugins → Add New → Upload Plugin**
3. Choose `osha-chat-widget.zip` and click **Install Now**
4. Click **Activate Plugin**

## Features

- 500+ OSHA regulation pages indexed
- Floating chat widget (bottom-right corner)
- Source citations with clickable links
- Mobile responsive
- Session history
- Safety orange theme

## Usage

Click the orange shield icon to ask OSHA safety questions like:
- "What are fall protection requirements?"
- "OSHA scaffolding regulations"
- "Confined space entry procedures"

## Configuration

Go to **Settings → OSHA Safety Chat** to:
- Change API endpoint (default: http://159.65.254.20:8080/chat)
- Select theme (Light/Dark)
